# Zenverse 项目

这是一个网页应用项目，提供用户下载移动端应用的功能。

## 功能说明

### 移动应用下载
- iOS 版本下载：通过 App Store 下载链接
- Android 版本下载：通过 Google Play 下载链接

## 项目结构
- `index.html`: 主页面文件
- `images/`: 存放项目相关图片资源
  - `app_store.svg`: iOS App Store 下载按钮图标
  - `google_play.svg`: Google Play 下载按钮图标
